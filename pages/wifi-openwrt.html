Getting Started:

SSH Key:  

Host:
Creating and ssh-key for the router.

ssh-keygen -t dsa

Next we will use the Secure Copy command to send the ssh-key from the host to the client to the OpenWRT router.

scp ~/.ssh/id_dsa.pub root@192.168.X.X:/tmp

Router:
Next we will log into the router to authorize the ssh-key

ssh root@192.168.X.X>

Change directory to dropbear (Still understanding dropbear. I'll post more when I look into to.)

cd /etc/dropbear

Next we will add the ssh-key to the to the list authorized keys acceptd.

cat /tmp/id_*.pub >> authorized_keys

chmod 0600 authorized_keys
