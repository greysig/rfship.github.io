Creating SSH Key:  

Host:
Creating and ssh-key for the router.

ssh-keygen -t dsa

Next we will use the Secure Copy command to send the ssh-key from the host to the client to the OpenWRT router.

scp ~/.ssh/id_dsa.pub root@192.168.X.X:/tmp

Router:
Next we will log into the router to authorize the ssh-key

ssh root@192.168.X.X>

Change directory to dropbear (Still understanding dropbear. I'll post more when I look into to.)

cd /etc/dropbear

Next we will add the ssh-key to the to the list authorized keys acceptd.

cat /tmp/id_*.pub >> authorized_keys

chmod 0600 authorized_keys

If I miss a step or don't show a step that you think is important, shoot me an email at rfship@gmail.com :)
Getting Started:
First we have to make sure that there are no process running that can get in the way.

airmon-ng check kill


Monitoring:
Next we will put out interface into monitoring mode. Since I am using a USB adapter my interface will be wlan1.

airmon-ng start wlan1


This will put the interface into monitoring mode. creating an new interface wlan1mon. With the new interface create we can begin to monitoring near by networks.



airodump-ng wlan1mon


The command above with begin listening to nearby networks processing the below output.

BSSID  -Mac address the Wireless Access Point
ESSID  - Name of the network. Can be hidden
PWR  -Signal Strength
CH  - Channel the Access Point is operating on
MB  -  Speed or mode the device is providing (11 pure)
ENC  - Encyption that is being used for the password
CIPHER  -Encryption protocol that is being using
AUTH  -Authentication being used to authenticate to the Access Point: PreShared Key (PSK)
Sniffing: 
Now that we have done a little monitor and found our target we can single them out and focus just one them

airodump-ng -c *Your target's channel* --bssid *bssid of target* -w /path/to/file --interface


Little bit of an explanation  -c wit be your target , 6 for me. -w simply mean write to then the path and name of file you wish to write out to. Finally you have the interface you are listening on. The command will begin to capture data. When you exit or quit the command it will output four files with various information.

        • *.cap
        • *.csv
        • *.kismet.csv
        • *.kismet.netxml
